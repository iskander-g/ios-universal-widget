# ios-universal-widget
> Simple universal widget for iOS (iOS more 8.0) based json format

## How to install
```pod install```
and ```run universalwidget.xcworkspace```

## How to use


## Received json format
```{  
   "title":"Title",
   "type":"list",
   "date":"23.12.2015 14:06",
   "data":[  
   		{  
         "name":"Подзаголовок 1:",
         "value":""
      },
      {  
         "name":"Данные 1",
         "value":"Значение 1"
      },
      {  
         "name":"Данные 2",
         "value":"Значение 2"
      }
   ]
}```
